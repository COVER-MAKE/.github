# AI를 활용한 도서관리 시스템

## 서비스 소개
AI 기술을 활용하여 도서의 내용을 분석하고 그에 어울리는 표지를 자동으로 생성해주는 스마트한 도서 관리 서비스입니다.</br>
누구나 쉽게 도서를 등록하고 관리하며 나만의 디지털 서재를 꾸밀 수 있습니다.

## 기술스택

### 백엔드
| 구분       | 기술      | 설명       |
|------------|----------|-----------|
| Language    | Java 17  | Backend 주 언어  |
| Framework   | Spring Boot 3.x  | 웹 애플리케이션 프레임워크     |
| Database    | H2 Database  | 개발용 인메모리 데이터베이스   |
| ORM | Spring Data JPA   | 객체 중심의 데이터 조작   |
| API   | REST API   | 자원 중심의 API 설계    |
| Tool  | Gradle | 빌드 관리 도구   |

### 프론트엔드
| 구분       | 기술      | 설명       |
|------------|----------|-----------|
| Language    | JavaScript (JSX)  | Frontend 개발 언어  |
| Library   | React 19.2.0  | 컴포넌트 기반의 UI 구축 라이브러리     |
| Build Tool    | Vite 7.2.4  | 프론트엔드 빌드 도구   |
| UI Framework | Material UI (MUI) 7.3.6   | 구글 Material Design 기반의 UI 컴포넌트 라이브러리   |
| HTTP Client   | Axios 1.13.2   | 비동기 서버 통신을 위한 Promise 기반 HTTP 클라이언트    |
